How We Calculate These Estimates

Server Count

Servers = Target Capacity ÷ Usable Capacity per Server

Minimum cluster size: 8 servers (VAST requirement)

Usable vs Raw Capacity

Usable ≈ 82% of Raw capacity

~18% overhead for VAST data protection (erasure coding)

Drive Configurations

  • • 15.3TB: 8 data drives + 2 NVRAM → ~100TB usable/server
  • • 30.7TB: 8 data drives + 2 NVRAM → ~200TB usable/server
  • • 61.4TB: 7 data drives + 3 NVRAM → ~350TB usable/server

Pricing

Switch Count

Switches = ⌈(Servers × 2 ports) ÷ 32⌉ rounded to pairs

Each server uses 4×200G ports (2×400G switch ports with breakout)
